# Frozen Ant Design theme assets

These icons, backgrounds, avatars, logo, and AlibabaSans font files are local copies used by the
theme preview. The Ant Design theme source baseline is commit
`9a5ca38c95b66764736853e4d0c4123484e69217` (Ant Design 6.5.1, 2026-07-14).

Runtime previews must reference these local files instead of the original CDN or GitHub avatar URLs
so visual baselines remain stable after upstream changes.
