## Description

实现 About 页面的「产品介绍」Tab 内容。展示 Theme Studio 的产品名称、一句话简介和核心功能列表（可视化编辑 Token、AI 生成主题、Playground 预览、主题导出、社区广场），使用 antd Typography、Card 等组件排版。

- [ ] Typecheck/lint passes
