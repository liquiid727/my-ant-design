# 更新 Ant Design 安装与主题接入模块

## Description

面向已经拥有可运行前端项目的用户，提供精简的 Ant Design 安装和 Theme Studio 主题接入内容。本模块不负责创建 React、TypeScript、Vite 或其他项目脚手架。

## Acceptance Criteria

- [ ] 展示 `npm install antd @ant-design/icons`
- [ ] 展示 Theme Studio `theme.ts` 和 `theme.json` 的导入方式
- [ ] 展示 `ConfigProvider theme` 接入示例
- [ ] 提供 Button、Card、Form 的主题验证代码片段
- [ ] 所有代码和命令支持一键复制
- [ ] 明确标注示例适用于 Ant Design 6
- [ ] 删除现有 `npm create vite` 内容
- [ ] 不包含 React、TypeScript、Vite、Next.js 初始化说明
- [ ] Typecheck 和 build 通过
- [ ] Verify in browser using dev-browser skill
